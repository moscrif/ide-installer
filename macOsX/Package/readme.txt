			README
			======

	    Moscrif SDK 2012q3a.157 release
	    ------------------------------

Welcome to the Moscrif SDK release!  
This release includes versions of MonoFramework-2.10.9_8.dmg and MoscriSDK.

Please see the FAQ at http://moscrif.com/faq?group=Instalation

Moscrif SDK Version notes
-----------------

Requirements
------------

Mac OS 10.7 or newer
Mono 2.10.4 or newer
1GB RAM
100MB of HDD

Note that Moscrif SDK requires Mono for Mac.  
Download the version from link below
http://moscrif.com/faq?group=Instalation&q=2

and install it in a suitable location.


Installation
------------

  MAC OS
  ----------

  1. Install prerequisites (MonoFramework-2.10.9_8.dmg).
     (You may have already done this.)

  2. Install MoscrifSDk.dmg
       
  3. Go to the Applications directory

  4. Run Moscrif IDE

Problems?
---------

Our web page at http://rc.moscrif.com/faq has a pointer
to the FAQ that include information on installation problems, etc.
